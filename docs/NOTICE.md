## Hershey vector font data (glyph coordinates)

The Hershey font coordinate data embedded in Hershey-Fonts and related JHF files derive from the historical Hershey / Usenet font distributions. When redistributing this software or derived builds, please retain the following acknowledgements (as requested by the original Usenet Font Consortium distribution terms, also referenced by Ghostscript Hershey documentation):

1. The Hershey Fonts were originally created by Dr. A. V. Hershey while working at the U. S. National Bureau of Standards.
2. The format of the font data was originally created by James Hurt, Cognition, Inc.
